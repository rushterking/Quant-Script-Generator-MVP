## 2. THE MATRIX HACK (Technical Architecture)
Ini bukan sekadar chatbot. Ini adalah **Text-to-Algo Engine** yang memanfaatkan High-Leverage API:
- [cite_start]**Core Intelligence**: Claude 3.5 Sonnet (via Blackbox AI API) - Model yang kebal terhadap halusinasi logika finansial[cite: 303, 422, 1092].
- [cite_start]**Frontend**: Streamlit (Zero-Latency UI) - Di-deploy pada infrastruktur serverless untuk skalabilitas absolut[cite: 67, 458].
- [cite_start]**Standard**: Menghasilkan kode **Pine Script v5** yang siap pakai di TradingView[cite: 588, 590, 1082].
